Objective:
To add songs to the database to start creating a playlist. 

Preconditions:
A database needs to be set to store songs.

Expected Outcomes:
- The user should be able to add songs to the database
- The songs added should be displayed in the database
- The user should be able to create a playlist and add songs

Test Results:
- Open the database and add songs
- The database will list out the songs added
- The user can create a playlist
- The new songs added can be search
